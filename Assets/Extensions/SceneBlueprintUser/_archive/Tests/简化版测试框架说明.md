## 简化要点

### 已移除
- CI/CD (GitHub Actions)
- Git Hook 脚本
- 复杂质量门禁系统
- 测试迁移工具

### 当前架构
- 3 个核心菜单：运行测试、测试配置、帮助指南
- 白名单程序集过滤（`TestRunnerFilter` + `TestConfiguration`）
- 测试完成后自动生成报告
- 一键复制报告给 AI 分析
