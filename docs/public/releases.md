# 发布与下载

SceneBlueprint 当前通过 GitHub Releases 提供公开下载入口。

## 下载入口

- [GitHub Releases](https://github.com/zgx197/SceneBlueprint/releases)

## 自动发布方式

- 推送符合 `v*` 规则的 Git tag 后，会自动触发桌面发布流程
- Release 会自动生成说明内容，并附加对应构建产物
- 当前正式发布以 Windows 桌面安装包为主

## 当前交付形式

- Windows 安装包
- Windows 桌面构建产物附件

后续当多平台发布成熟后，会继续在同一入口扩展更多平台产物。

## 版本类型

- 正式发布：通常由稳定版本 tag 自动触发
- 预发布：适合提前体验、验证与集成联调

## 版本命名

GitHub Release 使用统一版本标签，例如 `v0.1.0`。
